⚙️ Guide de configuration Google Drive API (cliquer pour développer)
1
Aller sur console.cloud.google.com → Créer un nouveau projet Parc Auto DRT Sfax
2
API et services → Activer Google Drive API et Google Picker API
3
Identifiants → Créer Clé API (restreindre au domaine) + ID client OAuth 2.0 type "Application Web" → Ajouter votre domaine dans "Origines autorisées"
4
Dans drive-integration.js, remplacer VOTRE_CLIENT_ID et VOTRE_API_KEY par vos vraies valeurs
5
Écran de consentement OAuth → Ajouter les utilisateurs autorisés (emails @tt.tn ou rendre l'app publique si nécessaire)
